缘起

14年8月从Inspur浪潮软件离职，本计划独自去川渝旅行一次，放松一下，可惜办理手续耽误时间较长.
提前三天来京找房子，临时定居西北旺，房子挺大，和哥们一起住，经常去爬百望山，吃米线.
后入职sogou，搬家，工作节奏瞬间飙升. 和之前完全不一样的工作节奏.
两个月后习惯了sogou的节奏，习惯了帝都的节奏.

14年前后，投资者目光转向了互联网。导致小创业公司如雨后春笋般冒出，许多开发团队都轻松拿到融资.
壕多了，他们财大气粗，用过分高的薪水去招人，去加快开发节奏，烧钱去推广，得到用户量后，继续融资.
融到B轮后，理论上就会有部分人赚到钱了，团队此时可以宣告失败. 继续开展其他项目，继续融资.
导致如我一般的码农嘴上骂着“就这狗屎般的app的竟然能给这么高工资，这是扰乱互联网市场的节奏”，
看着缺眼馋，也想趁机干他一票.

but…

故事开始

在浪潮软件的两年，做私有云平台. 当值”云计算，大数据”等概念被吹上了天. 当然，依然在吹，
现在卖狗屎的，都会说自己的是”云狗屎”, 哎，言论自由.

在过程中，学到了一些东西，比如LNMP, Vagrant
当时想做一个社交网站，积累点用户量，积累点经验，但是没有找到合适的方向.

后来到了sogou，做广告系统运维. 期间用CI做了一个小的管理系统，可惜没有用起来. 期间有想做娱乐app的想法.
但是不会Android，不会ios， 也不想投入时间去搞了， 就放弃了.

随后入我场, 工作中需用用到yaf，然不会，所以想用yaf框架来练练手,
此时，朋友的创业公司已经融到了A轮. 羡慕不已.
刚好又从微信看到外国一个婚恋网站卖了很高价钱的新闻.

相对而言，程序员算是非常公平竞争的职位，如果有实力就应该指着别的代码说”你写的这坨是啥！！”
所以，打算搞点什么，结束生活除了上班就是睡觉的生活.

搞起, 从html开始

没有规划， 没有美工， 没有CXO，只有周末和晚上下班的时间， 就开始搞起了.
在app盛行的时代， 我却开始搞起了web，只因为我不会搞app…
现在html+css+js的时代已经过去，已然是node bootstrap angular sass 等高大上的时代.
工作换进： win7电脑virtual box下安装了Centos， Centos上搭建了LNMP, putty + vim + filezilla.
目标：

眼高手低，这句话只有在真正动手的时候才能体会到.
一个简单的css样式，会花费半天的时间.
一个路由规则，可能会花费一天的时间都搞不定.
一个js特效，就必须百度、google到足够好的方法才会去用.
IE8，最终放弃了这货，可以判断如果是这货的时候，提示用别的方式浏览. twitter是这样做的.
一个合适的图片，都可能占用一个小时甚至更多的时间.
……

用了差不多两个月的时间，才做了首页给朋友看， 得到“这是啥啊”的回复.
重做， 又不忍心将已有的代码删除， 此时，才真正体会到一个svn的重要性。
用了两个小时，自己搭建了个svn，自己用，不需要考虑权限等问题.

putty + vim + filezilla + svn + bootstrap + gimp

再次起航，希望可以走到更远.

2015/09/05 阅兵后第二天
